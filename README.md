# Portfolio Website

This is my personal portfolio website, showcasing my projects, skills, and experience. The site is fully responsive and built using React with a clean and modern UI. Interactive elements and animations improve user engagement and accessibility.

## Features

- Responsive layout for desktop
- Interactive sections and animations
- Intuitive navigation for an engaging user experience
- Modern, minimalistic UI design

## Tech Stack

**Frontend:** React, TypeScript

**Styling:** CSS

**Hosting:** AWS S3 + CloudFront

**Security:** S3 bucket protected with Origin Access Control (OAC)

**Tools**: Vite, EmailJS, React Router

## Deployment

The website is deployed on **AWS S3** and served via **CloudFront**, providing:

- Secure HTTPS access

- Fast, globally distributed content

- Support for React Router via proper error handling configuration

Live URL: https://d1ib7vpujjcxv7.cloudfront.net/
